# aymm v0.1.1

Patch release: UX polish for the speech bubble and the Pomodoro Focus
entry, plus an explicit GNOME compatibility note and the first Ubuntu
26.04 binaries.

## What's new

- **Speech bubble** no longer overlaps the cat. The bubble is anchored
  higher and its tail tip lands just above the cat's ear tops, so the
  pointer reads as "coming from the cat" instead of sitting on top of
  its back.
- **Cat walks back to the basket.** When a Pomodoro break ends and the
  next Focus phase begins, the cat used to *teleport* into the
  configured corner. Now `focus_corner_position()` returns the target
  and `on_tick` walks the cat there at `pet.speed`, using the same
  state-machine transitions (`Wake` → `Run` → ... → `Sleep`) as the
  cursor chase. The cat literally trots back to its basket and curls
  up.
- **Ubuntu 26.04 LTS binaries** are now part of the release.

## Compositor compatibility (recap)

| Compositor               | Layer-shell overlay    | Cursor chase                       |
|--------------------------|------------------------|------------------------------------|
| Hyprland                 | ✅                      | ✅ via Hyprland IPC                 |
| KDE Plasma 6 / KWin      | ✅                      | ❌ libinput `EVIOCGRAB` exclusivity |
| sway / wlroots generic   | ✅                      | ⚠️ libinput grab, same as KDE       |
| GNOME / Mutter           | ❌ no `wlr-layer-shell` | ❌ overlay never opens              |

GNOME testing was added this release and confirms Mutter doesn't expose
`wlr-layer-shell-unstable-v1` — aymm exits at startup on GNOME with
`compositor missing required globals`. Workaround: install Hyprland
(`sudo apt install hyprland` on Ubuntu, `sudo dnf install hyprland` on
Fedora) and log into a Hyprland session.

## Quick start

```sh
tar xzf aymm-0.1.1-master-<distro>-x86_64.tar.gz
cd aymm-0.1.1-master-<distro>-x86_64
./bin/aymm --version          # verify hash
./bin/aymm                    # start the daemon in a Hyprland session
./bin/aymm --pomodoro         # one-command Pomodoro launch
./bin/aymm --help             # CLI reference
```

Run aymm **as your normal user, inside a Hyprland session terminal** —
not as root and not over plain SSH.
