# aymm v0.1.3

UX polish + first Arch + Debian binaries. The right-click cycle is more
playful, the speech bubble can finally render emoji, and the Pomodoro
notifications carry the cat icon properly across compositors.

## What's new

- **Emoji in the speech bubble.** v0.1.2 had to drop emojis from the
  bubble because Cairo's toy text API couldn't fall back to a color
  emoji font. v0.1.3 routes bubble text through Pango, so
  fontconfig-based fallback kicks in and the paw / cat / sleep emojis
  render as expected. The bubble copy is back to mood: e.g.
  `Break time, Queen 🐾`.

- **Random action selection (no repeat).** Right-clicking the cat used
  to walk through Ball → Food → Purr in fixed order. v0.1.3 picks the
  next effect at random from `{ Ball, Food, Purr, Scratch }` and
  refuses to play the same one twice in a row.

- **New action: Scratching post.** Wooden post wrapped in sisal rope,
  standing in front of the cat. Animated claw marks slide down the post
  during the ~4 s effect.

- **Yarn ball replaces the red ball.** Cream-coloured sphere with thread
  strokes wrapping around it and a small loose end. Reads as a cat toy.

- **Cat now stops in front of the ball.** The chase used to land the
  cat's centre exactly on top of the yarn — ball under the belly. The
  approach distance is now 40 px, so the ball stays visible in front of
  the cat.

- **Bowl heaped with food and placed in front.** Bowl moves to whichever
  side the cat is facing (east/west) and ships with two layers of
  overlapping kibble + a rim highlight. No more empty bowl.

- **Build outputs for Debian 13 and Arch / CachyOS** are part of the
  release.

## New build dependency: pangocairo

Existing builds need `pangocairo` available at configure time. Per-distro:

| Distro      | Package |
|-------------|---------|
| NixOS       | `pango` (the flake's `buildInputs` and `shell.nix` already include it) |
| Fedora      | `sudo dnf install pango-devel` |
| Ubuntu/Debian | `sudo apt install libpango1.0-dev` |
| Arch        | `sudo pacman -S pango` |

## Compositor compatibility (recap)

| Compositor                | Layer-shell overlay    | Cursor chase                       | Right-click play |
|---------------------------|------------------------|------------------------------------|------------------|
| Hyprland                  | ✅                      | ✅ via Hyprland IPC                 | ✅                |
| KDE Plasma 6 / KWin       | ✅                      | ❌ libinput `EVIOCGRAB` exclusivity | ✅                |
| sway / wlroots generic    | ✅                      | ⚠️ libinput grab, same as KDE       | ✅                |
| GNOME / Mutter            | ❌ no `wlr-layer-shell` | ❌ overlay never opens              | ❌                |
