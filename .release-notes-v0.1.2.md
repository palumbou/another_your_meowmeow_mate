# aymm v0.1.2

UX polish + first interactive features. Right-clicking on the cat now
triggers playful actions, the on-screen messages no longer pretend the
cat is talking while asleep, and system notifications carry a cat icon.

## What's new

- **Right-click the cat to play.** A new cycle of three interactive
  effects, each lasting ~3.5 s and triggered by clicking right-mouse on
  the cat (input region follows the cat around). The cycle resets after
  10 s of inactivity.
  - **Ball** — a red ball appears nearby, and the cat trots over to chase it.
  - **Food** — a brown bowl with food appears below the cat; cat sits and "eats".
  - **Purr** — the cat sits in place while three pink hearts drift up around it.

- **Status banner during Pomodoro Focus.** Entering a focus session no
  longer shows a speech bubble pointing at the cat (it's curling up to
  sleep, can't talk). Instead aymm shows a status banner centered above
  the cat, no tail.

- **No more tofu rectangles in bubbles.** Cairo's toy text API doesn't
  do font fallback for emoji, so the paw glyph rendered as a rectangle.
  Bubble strings are now ASCII-only. System notifications still use
  emoji freely — every notify-send body now starts with `🐈 ` so it's
  obvious it's from aymm.

## Compositor compatibility (recap)

| Compositor                | Layer-shell overlay    | Cursor chase                       | Right-click play |
|---------------------------|------------------------|------------------------------------|------------------|
| Hyprland                  | ✅                      | ✅ via Hyprland IPC                 | ✅                |
| KDE Plasma 6 / KWin       | ✅                      | ❌ libinput `EVIOCGRAB` exclusivity | ✅                |
| sway / wlroots generic    | ✅                      | ⚠️ libinput grab, same as KDE       | ✅                |
| GNOME / Mutter            | ❌ no `wlr-layer-shell` | ❌ overlay never opens              | ❌                |

The right-click cycle works wherever the layer-shell overlay does — KDE
users finally get an interaction, even without cursor chase.
