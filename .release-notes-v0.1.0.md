# aymm v0.1.0 — first public release

A Wayland-native desktop pet for Hyprland: cursor chase via Hyprland's
IPC socket, optional Pomodoro mode (CLI + config), Waybar custom-module
integration, multi-output overlay, and a custom Cairo-drawn cat
(including a pink-basket sleep pose). Optional Queen persona on the
`feature/queen-bianca-mode` branch.

`bin/aymm --version` self-identifies with the commit hash + commit date,
e.g. `aymm 0.1.0+d39862ea (2026-05-07)`.

## Quick start

```sh
tar xzf aymm-0.1.0-master-<your-distro>.tar.gz
cd aymm-0.1.0-master-<your-distro>
./bin/aymm                 # cursor chase
./bin/aymm --pomodoro      # one-command Pomodoro launch
```

Or build from source:

```sh
tar xzf aymm-0.1.0-master-source.tar.gz
cd aymm-0.1.0-master
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j
./build/aymm --version
```

## Compositor compatibility

| Compositor               | Layer-shell | Cursor chase                           |
|--------------------------|-------------|----------------------------------------|
| Hyprland                 | ✅           | ✅ via Hyprland IPC                     |
| KDE Plasma 6 (KWin)      | ✅           | ❌ libinput EVIOCGRAB exclusivity       |
| sway / wlroots generic   | ✅           | ⚠️ libinput grab, same as KDE           |
| GNOME Shell (Mutter)     | ❓ untested  | ❌ no public cursor API                 |

**Run as your normal user.** `sudo` / `su -` strip `WAYLAND_DISPLAY`
and `XDG_RUNTIME_DIR`, so the binary won't find the Wayland socket.
